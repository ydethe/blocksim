blocksim.graphics.FigureSpec module

class blocksim.graphics.FigureSpec.FigureSpec(props: dict, axes: blocksim.graphics.AxeSpec.AxeSpec)[source]

Bases : object

Class that provides a description of a figure, without data. It handles the axes layout, and the lines to be drawn (with the name of the variables instead of a concrete set of data)

Paramètres

  • props – A dictionary. Only key supported : title for the figure title

  • axes – List of AxeSpec to specify the axes” spec

Exemples

>>> fs = FigureSpec.specForOneAxeMultiLines([{'var':'th_mes','linestyle':'', 'marker':'+'}])
classmethod specForOneAxeMultiLines(line_list: Iterable[dict]) → blocksim.graphics.FigureSpec.FigureSpec[source]

Returns a FigureSpec to draw all the given variables on one same axe

Paramètres

line_list –

List of dictionary, whose keys are :

  • the matplotlib keyword arguments of the funcion plot

  • varx for the name of the X variable. If not specified, varx will be assumed to be the time variable “t”

  • var or vary for the name of the y variable

Returns

A FigureSpec that describes the layout
