blocksim.core.Generic module

class blocksim.core.Generic.GenericComputer(name: str, shape_in: tuple, shape_out: tuple, callable: typing.Callable, dtype_in=<class 'numpy.float64'>, dtype_out=<class 'numpy.float64'>)[source]

Bases : blocksim.core.Node.AComputer

Generic computer, that uses a vector function to transform an input

The input of the element is xin The output of the element is xout

Paramètres

  • name – Name of the element

  • shape_in – Shape of the input data

  • shape_out – Shape of the input data

  • callable – A callable object, use to compute the output

Exemples

>>> com = GenericComputer("com", shape_in=(5,), shape_out=(1,), callable=np.sum, dtype_in=np.int64, dtype_out=np.int64)
>>> out = com.compute_outputs(t1=0, t2=1, xin=np.arange(5), xout=None)
>>> out["xout"]
10
