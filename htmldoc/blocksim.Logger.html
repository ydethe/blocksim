getFilteredValue(name: str, ntaps: int, cutoff: float, window: str = 'hamming') → numpy.array[source]

Get the value of a logged variable, and applies a low-pass filter The argument can be an expression. It can combine several variables numpy functions can be used with the module name “np”: for example : np.cos

Paramètres

  • name – Name or expression

  • ntaps – Number of coefficients of the filter

  • cutoff – Cutoff frequency, normalized by the Nyquist frequency

  • window – Window function to apply. Default : hamming

Renvoie

An array of the values

Exemples

>>> ns=1000;fs=100;f0=10
>>> log = Logger()
>>> _ = [(log.log('t',t),log.log('s',np.cos(np.pi*2*t*f0))) for t in np.arange(0,ns)/fs]
>>> s = log.getValue('s')
>>> e = np.sum(s**2)/ns
>>> e
0.5...
>>> sf = log.getFilteredValue('s', ntaps=64, cutoff=20*2/fs) # 20 Hz filter
>>> ef = np.sum(sf**2)/ns
>>> ef
0.498...
>>> sf = log.getFilteredValue('s', ntaps=64, cutoff=5*2/fs) # 5 Hz filter
>>> ef = np.sum(sf**2)/ns
>>> ef
4.6...e-05
