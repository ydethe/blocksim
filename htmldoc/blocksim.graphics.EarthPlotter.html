blocksim.graphics.EarthPlotter module

class blocksim.graphics.EarthPlotter.EarthPlotter(projection=<cartopy.crs.PlateCarree object>)[source]

Bases : object

The methods of this class assume a cartopy compatile axe. To create such an axe, you can either use blocksim.EarthPlotter.EarthPlotter.createAxe:

    sp = EarthPlotter()
    axe = sp.createAxe(fig, 1, 1, 1)

or do it manually::

    axe = fig.add_subplot(111, projection=ccrs.PlateCarree())
    axe.stock_img()
    axe.gridlines(crs=ccrs.PlateCarree())
Paramètres

projection – The projection used for the map. See https://scitools.org.uk/cartopy/docs/latest/crs/projections.html
