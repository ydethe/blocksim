class blocksim.gnss.GNSSReceiver.GNSSReceiver(name: str, nsat: int, lon: float, lat: float, alt: float, tsync: datetime.datetime)[source]

Bases : blocksim.core.Node.AComputer

GNSS receiver. Warning: before any simulation, you should call initialisePosition

The input of the computer is measurements The outputs of the computer are realpos, estpos and estclkerror

The following parameters are to be defined by the user :

  • algo : Type of algorihtm to use. Can be “ranging” or “doppler”

Paramètres

  • name – Name of the element

  • nsat – Number of satellites flollowed by the tracker

  • lon (deg) – True longitude of the receiver

  • lat (deg) – True latitude of the receiver

  • alt (m) – True altitude of the receiver

  • tsync – Date where the calculation takes place
