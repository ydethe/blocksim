class blocksim.dsp.ADSPComputer.ADSPComputer(name: str, input_name: str = 'input', input_size: int = 1, input_dtype=<class 'numpy.complex128'>, output_name: str = 'output', output_size: int = 1, output_dtype=<class 'numpy.complex128'>)[source]

Bases : blocksim.core.Node.AComputer

Generic DSP Computer. It processes one input into one output. Adds a helper method process, that batch computes a set of input

Paramètres

  • name – Name of the computer

  • input_name – Name of the input

  • input_size – Size of the input vector

  • input_dtype – Type of the input vector

  • output_name – Name of the output

  • output_size – Size of the output vector

  • output_dtype – Type of the output vector

flatten(data: numpy.array) → numpy.array[source]

Given a batch block of input, returns a 1D array for processing The first column of the block is copied, then the second and so on

Paramètres

data – The block of data

Renvoie

The 1D copy

Exemples

>>> a = DummyDSPComputer()
>>> data = np.arange(6).reshape((3, 2))
>>> data
array([[0, 1],
       [2, 3],
       [4, 5]])
>>> a.flatten(data)
array([0, 2, 4, 1, 3, 5])
