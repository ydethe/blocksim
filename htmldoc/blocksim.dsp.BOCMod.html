blocksim.dsp.BOCMod module

class blocksim.dsp.BOCMod.BOCMapping(name: str, f_ref: float, m: int, n: int, p_samp: int = 10, input_size: int = 1)[source]

Bases : blocksim.dsp.ADSPComputer.ADSPComputer

Returns a BOC sequence that, once convoluted with a PRN, gives the modulated signal Sub carrier frequency and chip frequency are given as multiples of f_ref = 1.023 MHz

Paramètres

  • name – Name of the computer

  • f_ref (Hz) – Chip rate of the modulation

  • m – Sub-carrier frequency multiplier : f_s = m.f_ref

  • n – Chip frequency multiplier : f_s = m.f_ref

  • p_samp – Muliplier so that the sequence is sampled at f_samp = p_samp*f_s

  • input_size – Number of bits modulated in parallel.
